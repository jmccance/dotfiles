jmccance.dotfiles
=================

## Requirements

- zsh
- [powerline](https://github.com/powerline/powerline)
- [stow](https://www.gnu.org/software/stow/)

## Supported Tooling

- [sdkman](http://sdkman.io/)
- tmux
  - [tpm](https://github.com/tmux-plugins/tpm)
- vim

## Setup

```
git checkout $REPO_URL ~/.dotfiles
cd ~/.dotfiles
./init.sh
```
